# Client

React-based frontend application for task management.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
npm run dev:development
npm run dev:production
```

## Build

```bash
npm run build
npm run build:development
npm run build:production
```

## Preview

```bash
npm run preview
npm run preview:development
npm run preview:production
```

## Technologies

- React 18
- TypeScript
- Material UI
- Zustand
- React Hook Form
- Zod
- Axios
- Vite
- date-fns

## Features

- Responsive design
- Pagination
- Task search with debounce
- Task creation, editing, and deletion
- Task completion toggle
