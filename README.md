# Task Management Application

A full-stack task management application built with React and Express.

## Technologies

### Client
- React 18
- TypeScript
- Material UI
- Zustand
- React Hook Form
- Zod
- Axios
- Vite
- date-fns

### Server
- Node.js
- Express
- TypeScript
- TypeORM
- PostgreSQL
- Swagger
- Docker
